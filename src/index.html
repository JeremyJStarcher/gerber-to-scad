Gerber to OpenSCAD Converter

Based on the powerful TraceSpace parser.

Warning: Experimental!

This program is an attempt to convert a Gerber file into an OpenSCAD file that is suitable for 3D printing. Currently, there are several features of Gerber files that are not supported, including its macro abilities.

So far, it has only been tested with KiCad Gerber files, as that is the tool that I use.
